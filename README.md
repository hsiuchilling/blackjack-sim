# blackjack-sim

Blackjack simulator for recreational observations.

## Getting Started

We use `poetry` as a package manager. Installation instructions are [here](https://python-poetry.org/docs/#installation). 

```
cd blackjack_sim
poetry install

poetry run python basic_play.py
```
